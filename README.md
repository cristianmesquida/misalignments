# Repository
This repository contains the manuscript, the figures and code used to simulate data and create the figures reported in the manuscript.

# Project sctructure:
```{r}
misalignments/
├── manuscript/
|   ├── manuscript.qmd      # Quarto document for the reproducible manuscript
│   ├── references.bib      # Bib file with the references used in the document
|   ├── preprint.pdf        # PDF document containing the preprint of the study
|   └── _extensions         # Quarto extensions for `manuscript.qmd`  
├── figures.Rmd             # R Markdown script to reproduce figures 2-5
└── figures/                # Contains figures reported in the manuscript
```
